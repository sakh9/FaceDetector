Simple Webcam Recognition With Tensorflow and Opencv
-----------------------------------------------------

## Description
A realtime object recognition using webcam, opencv, and Tensorflow (w/ Keras).
Using Teachable Machine as a Based Model, and improving it through many references.

## Fitur
- Realtime webcam capture
- Image classification using TensorFlow Keras
- Confidence score for every prediction
- Simple optimization for realtime performance

## Installation and Configure Using *bash*
1. Clone repository
```git clone <repo-url>```
```cd project```

2. Create Virtual Environment
```python -m venv .venv```

3. Install Dependency
```pip install -r requirements.txt```

## Run Program
```python main.py```

Click ```ESC``` to exit the program
